# 🏠 House Price Prediction Web Application

A Machine Learning-based House Price Prediction system built with:

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- API Communication: REST API (JSON)

Users can enter house details such as income, age, rooms, population, etc., and receive a predicted house price.

---

## 📌 Features

✅ User-friendly Interface

✅ Real-time Price Prediction

✅ REST API Integration

✅ Express.js Backend
